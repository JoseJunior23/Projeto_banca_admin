### Clone do projeto
``` bash
#clonar o repositório
$ git clone https://github.com/JoseJunior23/banca_Admin

# Entrar no diretório
$ cd banca_Admin

#Instalar as dependências
$ yarn install
      ou
  npm install

#Iniciar o projeto 
$ yarn start
      ou
  npm run      

```
